好友信息 API 模式

好友关系与见面记录

使用 http://api.tczhong.com/friend 管理好友列表，追踪最近一次见面，记录备注，并随时同步到 Dashboard。

好友列表

支持按姓名过滤，并根据见面时间或姓名排序。

创建好友

姓名为必填，其他字段可选；last_meet 支持 ISO 或 “2024-06-01 20:30:00+08” 格式。

更新好友记录

只填写需要变更的字段即可；notes 和 last_meet 均为可选。

查询单个好友

输入记录 ID，快速查看详细备注。

提交 ID 后将在此展示 API 返回的数据。

常用 cURL 调试参考
# 列出好友
curl -X GET "http://api.tczhong.com/friend?order=desc&order_by=last_meet"

# 创建好友
curl -X POST "http://api.tczhong.com/friend/" \
  -H "Content-Type: application/json" \
  -d '{"name":"Alice","relationship":"同事","last_meet":"2024-05-12T19:00:00+08:00"}'

# 更新好友
curl -X PUT "http://api.tczhong.com/friend/{id}" \
  -H "Content-Type: application/json" \
  -d '{"notes":"2024-06-01 约晚餐","last_meet":"2024-06-01T20:30:00+08:00"}'
